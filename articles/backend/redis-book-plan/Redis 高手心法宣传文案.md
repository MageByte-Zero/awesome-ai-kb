### 内容亮点

本书全面覆盖了 Redis 从基础到高阶的所有知识点，深入剖析了 Redis 的底层实现原理，包含了 7.0 版本的各种全新特性。以下是本书的章节概要：

- **起势入门**：带你构建一个源码可调式开发环境，从一条命令的执行开始，勾勒出 Redis 的数据存储原理和整体架构。
- **核心筑基，数据结构与心法**：详细讲解 Redis 的各种数据类型及其底层实现原理，包括 String、Lists、Sets、散列表、Sorted Sets、Stream、Geospatial、Bitmap、HyperLogLog、Bloom Filter 等。
- **不死之身高可用**：深入探讨 RDB 快照、AOF、主从复制架构、哨兵集群和 Redis 集群的实现原理。
- **结丹飞升高级技能进阶**：介绍 Redis 事务、内存管理、事件驱动框架、发布/订阅机制实现原理，内存碎片深度解析与优化策略，客户端缓存和 I/O 多线程模型原理。
- **元婴大成出师实战**：包括性能排查与解决问题的检查清单、使用规范、内存优化技巧、生产王者必备配置、缓存使用策略、缓存策略与数据库一致性问题深度剖析和分布式锁演进原理。

### 独特卖点

本书从 Redis 的视角与各路“神仙”对话，探讨每个技术点的原理，并以生动形象且准确的图片、诙谐幽默的语言来解释难懂的技术点。每一个章节都经过反复推敲，只为呈现最精彩的内容。

经过多年对 Redis 的深耕，我重新梳理了 Redis 技术架构，加入了 Redis 6.0 ~ 7.0 版本的各种全新特性，从更深层次的角度挖掘底层实现原理。我站在开发者的角度，用拟人化、场景化诙谐幽默的言语，再加撩人心弦又准确的图片，让读者轻松愉快地学习 Redis 实现原理和开发实战技巧。

- **系统架构师**：从全局视角掌握 Redis 架构和原理，学习 Redis 高可用、高性能的设计思想，解决 Redis 性能难题。
- **刚入行的程序员**：不想仅停留在“面试八股文”的阶段，更深层次地掌握 Redis 内部原理和实战技巧，在面试或者工作中脱颖而出。
