## 欲练此功不必自攻只需放松

将复杂的概念与实际案例相结合，让理论原理与实战相结合，以简洁诙谐幽默的方式配以大量撩人心弦又准确的图片，为你揭示 Redis 的精髓。哪怕是“天阶高级”斗技，也能愉快的修炼，而不会走火入魔。

##  知其然知其所以然

站在开发者的角度，提炼出 Redis 核心技术并归纳总结，让你所学都是必杀技，付出都将回报。

第一章：构建一个源码可调式开发环境，从源码目录结构，一条命令的执行开始，勾勒出 Redis 的数据存储原理和整体架构，构建全局视角，避免一叶障目。

第二章：核心筑基——结合实战掌握每种数据类型的使用技巧和场景，以及这些数据类型的底层数据结构实现原理（SDS、linkedlist、ziplist、quicklist、listpack、intset、dict、skiplist、Radix Tree、Bitmap、HyperLogLog 等）。 

第三章：不死之身——高可用，RDB 快照和 AOF 实现数据持久化、写时复制技术的运用、Multi-PartAOF 机制、主从复制架构、哨兵集群、Cluster 实现原理。

第四章：结丹飞升——高级技能进阶，Redis 如何实现事务、内存管理机制（淘汰策略原理、过期删除策略）、事件驱动框架 aeMain、发布/订阅机制原理与实战、客户端缓存、 I/O 多线程模型、内存碎片解析和优化。

第五章：性能排查与解决方案的终极检查清单、开发和运维使用规范、内存优化必杀技、生产必备配置详解、缓存使用策略、缓存击穿、缓存穿透、缓存雪崩解决方案、数据库与缓存一致性原理分析、分布式锁演进原理与实战。
